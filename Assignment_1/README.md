# Python Assignment – Basic Operations & Greeting Program

This repository contains two Python scripts created for the assignment.

## Task 1: Basic Mathematical Operations
- Takes two numbers as input.
- Performs:
  - Addition
  - Subtraction
  - Multiplication
  - Division (handles division by zero)
- Displays the result of each operation.

## Task 2: Personalized Greeting
- Takes first name and last name as input.
- Combines them into a full name.
- Prints a personalized greeting message.

## Files Included
- task1.py – Code for mathematical operations
- task2.py – Code for personalized greeting
- README.md – Description of the project
